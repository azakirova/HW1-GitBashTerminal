XML
 21. Создать внешний репозиторий c названием XML.

 22. Клонировать репозиторий XML на локальный компьютер.

 23. Внутри локального XML создать файл “new.xml”.
$ cat > new.xml

 24. Добавить файл под гит.
$ git add new.xml

 25. Закоммитить файл.
$ git commit -m "create new.xml"
[master 1438246] create new.xml
 1 file changed, 0 insertions(+), 0 deletions(-)
 create mode 100644 new.xml

 26. Отправить файл на внешний GitHub репозиторий.
$ git push

 27. Отредактировать содержание файла “new.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML.
$ cat > new.xml
<?xml version="1.0" encoding="UTF-8" ?>
<Information_about_me>
    <lastname>Zakirova</lastname>
    <name>Anastasia</name>
    <patronimyc>Andreevna</patronimyc>
    <age>28</age>
    <pets>Dog boxer</pets>
    <desired_salary>100 000 rub</desired_salary>
</Information_about_me>

 28. Отправить изменения на внешний репозиторий.
$ git push

 29. Создать файл preferences.xml

 30. В файл preferences.xml добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате XML.

$ cat > preferences.xml
<?xml version="1.0" encoding="UTF-8" ?>
<preferences>
    <favorite_film>Pride and Prejudice</favorite_film>
    <favorite_serial>Game of thrones</favorite_serial>
    <favorite_food>Steak</favorite_food>
    <favorite_season>Summer</favorite_season>
    <favorite_country>Spain</favorite_country>
</preferences>
 
31. Создать файл sklls.xml добавить информацию о скиллах которые будут изучены на курсе в формате XML

$ cat > skills.xml
<?xml version="1.0" encoding="UTF-8" ?>
<skills>
    <Git_Bash>interaction with github</Git_Bash>
    <Postman>testing API</Postman>
    <Charles_and_Fiddler>Get logs</Charles_and_Fiddler>
    <Mobile_testing>iOS, Android, guideline, ADB, application assembly</Mobile_testing>
    <SQL>SQL Basics, Postgres database</SQL>
    <Jmeter>stress testing</Jmeter>
</skills>

 32. Сделать коммит в одну строку.
$ git add preferences.xml skills.xml
$ git commit -m "preferences.xml skills.xml"

 33. Отправить сразу 2 файла на внешний репозиторий.
$ git push
